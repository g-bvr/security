WithSecretCommand

Purpose

Execute a command using a secret

Variants

UsageDescription
WITH SECRET encrypted IN VARIABLE variable *store decrypted secret into variable before executing the command, deletee variable afterwards
WITH SECRET encrypted IN FILE decrypted *store decrypted secret into file before executing the command, deletee variable afterwards

Arguments

ArgumentValue
encryptedthe path to the file with the encrypted secret data (relative to current workspace)
decryptedthe name of the file with decrypted secret data (this is not stored in workspace, but stored in a separate special folder)
variablethe name of the variable into with the decrypted secret data is stored
*the command to be executed (the command can use the decrypte secret, but it is deleted again after command execution)

Plugin

This command is defined in SecurityPlugin
