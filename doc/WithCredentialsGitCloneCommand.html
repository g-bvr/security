WithCredentialsGitCloneCommand

Purpose

clone a git repository using encrypted credentials

Variants

UsageDescription
GIT WITH CREDENTIALS credentials CLONE baseurl repositoryclone the default branch of the repository
GIT WITH CREDENTIALS credentials CLONE baseurl repository tagclone the specified branch or tag of the repository

Arguments

ArgumentValue
credentialsPath of encrypted credentials file (relative to current workspace)
baseurlThe url prefix of the repository (not including the actual repo name)
repositoryThe name of the repository (which together with the base url constitutes the url of the repository)
tagOptional argument to specify either a branch or a tag which shall be checked out (if omitted the default branch will be used)

Plugin

This command is defined in SecurityPlugin
