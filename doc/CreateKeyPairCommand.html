CreateKeyPairCommand

Purpose

create a new public/private key pair and execute a command

Variants

UsageDescription
WITH NEW MASTER KEY SIZE keysize BITS DO *create a new public/private key pair and execute a command

Arguments

ArgumentValue
keysizenumber of bits to use for asymmetric/symmetric key, format: #,#
*command to be executed, the name of the file with the created key can be referred to with %
